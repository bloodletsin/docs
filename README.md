# Welcome to Bleh

A premium Discord bot giving you the resources to create a powerful and versatile community at the tip of your fingers.

## Getting Started

- **Server prefix** is set to `;` by default. Use `;prefix set (symbol)` to change it.
- Type `;help` to see all available commands.
- Type `;help <command>` for details on a specific command.
